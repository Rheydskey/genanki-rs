## Genanki-rs
Generates cards from a GIT repo containing markdown source.

## Structure of a genanki-rs project
```
/
|- deck_name1/
      |- 1.md
      |- 2.md
      |- ...
|- deck_name2/
      |- 1.md
      |- 2.md
      |- ...
|- .../
```

## Example
deck_name/afile.md
```md
## Give me the definiton
This is the definition
```

The deck named `deck_name` will contains one card.
